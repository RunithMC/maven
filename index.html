Runith Maven Repository

Public artifacts for the Runith api,

https://repo.runith.net/

Gradle (Kotlin DSL)

Add the repository

repositories {
    maven {
        url = uri("https://repo.runith.net/")
    }
}

Gradle (Groovy DSL)

repositories {
    maven {
        url "https://repo.runith.net/"
    }
}

Maven

Add the repository

<repositories>
    <repository>
        <id>reactor-maven</id>
        <url>https://repo.reactor.codes/</url>
    </repository>
</repositories>

Important Notes

  • The Runith API is intended to be used as a compile-time dependency.
  • No authentication is required to access this repository.
